ORPHE-CORE.js Tutorial

このチュートリアルページを読む前に必ずdocumentページのQuick Start with VSCodeまでを読み，サンプルプログラムの実行環境を整えてください。手元で動作させながら学習することでORPHE-CORE.jsの理解が深まります。

ORPHE-CORE.js Tutorialは ORPHE-CORE.js を初めて扱う方向けのチュートリアルです。ドキュメントページでまずは使ってみることを体験した後、アプリケーション開発時に必要となるコアモジュールに関する基礎的知識やORPHE-CORE.jsの基礎理解を深めることを目的としています。以下のチュートリアルは番号順に学習することを強くおすすめします。

  1. ORPHE-CORE.js Basic

    ORPHE-CORE.jsとBLE通信における基礎概念とOrpheクラスの理解を深めます。

  2. CoreToolkit.js Basic（準備中）

    CoreToolkit.jsの基本的な使い方を学びます。CoreToolkit.jsはORPHE-CORE.jsのコアモジュールを使いやすくするためのユーティリティモジュールです。
