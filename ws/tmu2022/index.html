YOU ARE the BIRD

YOU ARE the BIRDは、ORPHE CORE moduleをコントローラーとしたフィットネスゲームです。プレイヤーはモジュールを手に持って羽ばたく動作をして、鳥のキャラクターを操作して障害物を避けます。鳥の羽ばたく力はモジュールの加速度センサーに対応しており、プレイヤーが大きく速く羽ばたく動きを行うほど、鳥を高く飛ばすことができます。プレイヤーはゲームを楽しみながら肩こりを解消することができます。

Virtual TonTon

ばーちゃるとんとんは、私たちが身近に遊べるとんとん相撲を発展させたゲームです。ゲーム上には土俵と２人の力士がおり、実際に行われたとんとん相撲の勝敗をそのゲーム上にも反映させ、演出を加え遊び自体のスケール感を増加させます。ORPHE COREをそれぞれとんとん相撲の力士型のハードウェアに装着し、OSC通信を用いてゲーム上にデータを送ります。ゲーム上の合図とともにとんとん相撲を始めると、ORPHE COREからAngle のデータが送られ、条件を満たすとゲーム内で転倒を検知し、勝敗の演出が流れるようになります。ゲーム上で力士の相撲中のアニメーションはありますが、移動はしません。実際のとんとん相撲の力士の位置も取得できたら、よりリアルでシュールなものになると思うので、土俵自体にセンサーを置いたり、上からカメラで認識してみたりと、今後検討していきたいです。

Long Jump Sonification

走幅跳のソニフィケーションを行う装置です。
最初に自分の走り始めのルーティンを登録します。ルーティンの登録中はルーティンボタンがピンク色になります。横のバーは右側の方がルーティンの類似度の許容範囲が広くなり、左にするほど狭くなります。なかなか動かない時は右側、すぐに動いてしまう時は左側にスライドすることをおすすめします。
その後、通常通り走幅跳の練習を行うことで自分の走幅跳の音を聞くことができます。また、波形は3回分保存され、[1][2][3]キーを押すことでその音を聞くこともできます。
この装置は自分の腰に装着して使用します。また、音は加速度を周波数に変換した音となっています。このソニフィケーションが走幅跳のフィードバックに追加されることで、リズムやピッチがよりわかりやすくなることはもちろん、上手な人と自分を比較する手段の一つにもなるはずです。

ORPHE COREでお掃除を楽しくするアイデア

ORPHE-COREを小学校のお掃除に導入することで、「掃除の時間がつまらない」という問題を解決することができる。また、教員・オルフェ側にも良い効果がある。
小学校のお掃除は主にほうきとぞうきんによって行われるので、ほうきの柄およびぞうきんがけをする人の手首にORPHE-COREを装着すれば、お掃除に関わる動作をスコア化できると考えた。ほうき担当の人においては、掌に逆さのほうきを直立させてバランスをとりほうきの直立を維持するいわゆる「ほうきバランスゲーム」のタイムをその人のスコアとする。ぞうきん担当の人においては、スワイプした回数をその人のスコアとする。ほうきバランスのタイム測定およびぞうきんのスワイプ回数はほうきが倒れた際のモーション並びにぞうきんスワイプのモーションをあらかじめ設定しておき、Dynamic Time Warpingによって行う。ゲームを行うとき以外においても、ORPHE-COREによってお掃除のパフォーマンスを取得することができるので、忙しい教員の負担軽減につながる。また、お掃除の時間を入口として、オルフェはORPHE-COREを小学校のプログラミング教材にするチャンスを得ることができる。

Ripples

水面を叩くと波紋が広がって行き、水の動きが可視化され、生成された波紋はどこかにぶつかると消えてなくなってしまう。
一方通行で波紋が消えてなくなるだけではなく、人が水面に干渉すると水面がそれに共鳴するように音を奏でる体験ができるシステムを制作した。
自分で波紋の源を動かすことで擬似的な水面と干渉し合い、ランダムなリズムが作り出され新たな表現ができるのではないかと考えた。まずORPHE COREを内蔵したこのデバイスは、上からカメラによって位置が割り出されており、ディスプレイ上にはデバイスと同じ座標にオブジェクトが表示されている。そしてデバイスを台の上に複数個置いてそのデバイスを自由に配置したのち、任意のデバイスを叩くと、PCのディスプレイに表示される対応したデバイスの位置から波紋と同時に音が鳴る。
波紋はディスプレイ上の他のデバイスの位置に当たるとそのデバイスからまた音と波紋が生成される。

チェンソーマンの腕

授業内容でOrpheを振ると、加速度センサーが検知する値により音を出す仕組みを学びました。何かを振ると音を出せる物で提案したのはアニメの武器でした。例えば、「鬼滅の刃」のアニメなどの人気が急上昇し、キャラのコスプレのためにおもちゃの刀を利用する子供が増加しました。ネットで検索したところ、ボタンで音が発生する刀のおもちゃはありますが、振ると同時に音が出るのはありませんでした。ですので、今回はおもちゃの武器を振ると音が鳴る商品を提案しました。
私は「チェーンソーマン」のアニメが好きなので、今回はチェーンソーマンの片腕のチェーンソーを作りました。素材は段ボールで作られています。腕をはめるには、穴の両端の部分に中指と薬指の谷と肘をフィットさせます。Orpheはチェーンソーの真ん中の位置に設置されており、振ると鳴る仕組みになっています。よりチェーンソーを表現するためにチェーンが動くよう制作しました。Arduinoを使用し、ボタンを押すと両端のモーターで紐のチェーンが交互に回転するようプログラムされています。

ORPHE COREを用いた文字体型の分析

現在、世界ではさまざま種類の文字が使われています。私はORPHE-COREを用いて、現用の文字体系について文字同士に新しい傾向を見出そうと試みました。書き文字は、その歴史的な文脈からの考察は既にいくつも行われていますが、動作による文字体系の分類はまだ行われていません。そこで、ORPHE-COREを用いることによって、筆記の動作的なアプローチから文字を考えることも可能なのではないかと考えました。
ORPHE-COREを指に取り付け、日本語、英語、フランス語、ロシア語、中国語、ハングル、ヒンディー語、アラビア語　の八種類の文字の筆記を行い、加速度の推移のグラフを観察しました。そして、そこから見えてきた各言語の文字の傾向を現用の文字体系の分類との比較しながら考察を行いました。

Orphe-ping

Orphecoreを消しゴムに見立てて消しピンを遊ぶためのUIと、それに伴うギミックを作成しました。ただし通常の消しピンとは異なり個人戦ではなく、Orphecore vs その他の消しゴムで戦う非対称型対戦ゲームです。
Orphecoreを接続し、画面上に[boss turn]と表示されているときはOrphecoreのターン、[your turn]と表示されているときは消しゴム側のターンです。turnボタンで切り替えてからはじいてください。またresetボタンでHPなどの数値をリセットできます。
OrphecoreのHPは500に設定されており、消しゴムにぶつかられた衝撃に応じて画面上のHPが減っていきます。HPが0になれば消しゴム側の勝ち、ほかの消しゴムをすべておとせばOrphecoreの勝ちとなります。Orphecoreに関しては素早く回すことによる音や光のギミックもあります。

NOD

「裏アカがバレそうなとき、その場で全く手を動かさずにツイートして、アリバイを作りたい！」
「両手が荷物でいっぱいだけど、次の曲に切り替えたい！」
「これ、言い過ぎじゃない？パワハラじゃない？上司にバレないようにボイレコをONにしておきたい！」
手が使えないとき、声も出せないときにも、緊急事態はやってくる。そんなとき力になるのは、誰にもバレずに操作ができる、チョーカー型ウェラブルインターフェース「NOD」。
飲み込む動き、咳き込む動き、発声の動きなど、登録されたあなたの喉の動きを検知して、
SNS投稿、プレイリスト再生、タイマーON/OFF、ボイスレコーダーONといった、役立つ機能を自動で起動させます。
喉の動きデータと機能の紐付けは自由にカスタマイズ可能。喉の動きデータは無限に増やすことができます。NODがあれば、スパイにだってなれる。
喉のチカラで、未来を救うのだ。

秘匿インタフェース

ORPHE CORE の小さく、無線通信で精度が高いという3つの特徴を活かして秘匿インターフェースを作りました。特に、ORPHE CORE の小さな動きを検知できるという特徴を活かしたかったので、筋肉や筋のつながりに着目し、動かすつもりでなくとも連動して動いてしまう筋肉の微小な動きを用いて動作の検知をすることにしました。目尻や喉、首の付け根など様々な部分を試し、最終的に腕頭骨筋にORPHE CORE を取り付けることにしました。腕頭骨筋は、手や指を動かしたときに連動して動いてしまう腕の筋肉です。秘匿インターフェースが生きる場面を考え、プレゼンテーションの際のスライドを操作の対象に選びました。ORPHE CORE でのスライド操作を実現するために、専用スライドアプリケーションも同時に作成しました。軽く手を握るとスライドが次のページに進み、親指と人差し指を擦ると前のページに戻ります。動作の検知はDTWで行なっています。

NINJA-clock!

【説明】
「NINJA-clock!」は、デスクワークを補助する機能を備えたアプリです。
ORPHE-core moduleを取り付けられる「ORPHE手甲」を装着した状態で使用します。WindowsおよびMacOSで動作するUnityで制作されたデスクトップアプリケーションで、
本アプリを起動すると画面上に忍者が出現します。「NINJA-clock!」の機能は、主に二つです。①タイマー機能
任意の時間を設定し、タイマーを作動させることができます。
タイマーはカウントダウンで減っていき、残り0秒になるとアラームが鳴ります。
アラームが鳴り始めると同時に、停止するためのコマンド「停止の印」が記載されたUIが出現します。
入力開始の印を結び、停止の印を結ぶことに成功するとアラームが停止します。②BGM再生機能
特定の印を結ぶことに成功すると、環境音を再生することができます。
画面上の「忍術一覧」から、結びたい術の印を確認することができます。
